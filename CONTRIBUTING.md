## 🐛 Reporting Bugs

If you find any bugs or endpoints that no longer work with Discord's undocumented Display Name API:

1. 📂 Locate the `.jsonl` log file in the `logs/` directory
2. 🔴 **Redact any bot tokens** from the log file
3. 💬 Join our [Discord server](https://discord.gg/UFCdsaMnYm)
4. 📝 Provide the sanitized `.jsonl` file with details about the issue

> 💡 The `.jsonl` files contain raw API request/response traces that help us diagnose problems quickly!
